# Body Fat Calculator

A precise body fat calculator using the scientifically validated U.S. Navy method. Calculate your body fat percentage, fat mass, lean mass, and BMI with detailed health interpretations.

🌐 **Live Website**: https://baslia.github.io/bodyfat.github.io/

## Features

- **Dual Unit System**: Switch between Metric (cm/kg) and Imperial (in/lb)
- **Two Calculation Modes**:
  - **Simple Mode**: Quick calculation using height, weight, neck, and waist measurements
  - **Advanced Mode**: Enhanced accuracy with additional measurements (chest, bicep, forearm, thigh, calf)
- **Gender-Specific Formulas**: Accurate calculations for both male and female body compositions
- **Interactive Measurement Guides**: Visual SVG illustrations showing exactly how to take each measurement
- **Comprehensive Results**: 
  - Body fat percentage
  - Fat mass
  - Lean body mass
  - Body Mass Index (BMI)
  - Health category and interpretation
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices

## How It Works

The calculator uses the U.S. Navy body circumference method, which is one of the most accurate non-invasive methods for estimating body fat percentage. The advanced mode incorporates additional measurements to refine the estimation further.

## Usage

Simply visit the website, select your gender, choose your preferred units and mode, enter your measurements, and click calculate. Use the help buttons (?) next to each measurement field to see visual guides on proper measurement techniques.

## Technology

Built with pure HTML, CSS, and JavaScript - no frameworks required. Fast, lightweight, and works offline once loaded.
